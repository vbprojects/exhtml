VPML VPML - A Simple HTML Template Language

Learn VPML

VPML (Very Personal Markup Language) is a simple templating language that compiles into HTML. It uses indentation to determine element hierarchy. The language is meant to be flexible, predictable, and easy to read. The repo also comes with tooling for compiling and using General Preprocesser (GPP) to serve as a preprocessor for VPML files.

Syntax

  • Tokens are delimited by spaces ' '.
  • Newline '\n' and carriage return '\r' characters are stripped from tokens.
  • Lines whose first token is a valid HTML tag name are processed as HTML elements.
  • Tokens starting with '#' within a line are considered as part of the ID for the element.
  • Tokens containing '=' within a line are processed as attributes with their respective values.
  • Tokens that do not fit the rules for ID or attributes are assumed to be class names.
  • Lines that do not start with a valid HTML tag or start with 'raw' are processed as text content.
  • Comments are denoted by a line starting with '?'.

Example 

head
  title
    VPML - Example
  link rel="stylesheet" href="styles.css"
html
? this is a comment
    body #body bg-black
        div #content bg-gray-800 #1
            h1 bg-blue
                Hello World
            p
?           this is a comment
                raw This is my site
        div #content bg-gray-800 #2
            h1 bg-blue
                Hello World

Output 

< head >
  < title >
    VPML - Example
  < / title >
  < link rel="stylesheet" href="styles.css" >
  < / link >
< / head >
< html >
    < body id="body" class="bg-black" >
        < div id="content1" class="bg-gray-800" >
            < h1 class="bg-blue" >
                Hello World
            < / h1 >
            < p >
                This is my site
            < / p >
        < / div >
        < div id="content2" class="bg-gray-800" >
            < h1 class="bg-blue" >
                Hello World
            < / h1 >
        < / div >
    < / body >
< / html >
